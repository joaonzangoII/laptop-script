laptop-script
=============

Borrowed from [thoughtbot/laptop](https://github.com/thoughtbot/laptop)

### Installation
Linux support only
Ubuntu Distros supported 
 - wheezy
 -jessie
 -precise
 -trusty
 -utopic
 -vivid
 -wily

```shell
$ bash <(wget -qO- https://raw.githubusercontent.com/joaonzangoii/laptop-script/master/linux) 2>&1 | tee ~/laptop.log
```
